== How To Use

Payload schema:
[source,javascript]
POST /render?componentLibrary=@coursehero-components/hello-world&component=Greeting
Content-Type: application/json
CH-Auth: ...
{
  "name": "World"
}

Response (raw):
[source,html,subs="macros"]
var:helloWorldResult[raw=true]

Response (rendered):
var:helloWorldResult[]
